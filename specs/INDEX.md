## Specs

- [x] 001 — hull-module — full track — Parametric Storebro hull geometry (LOA, beam, draft, deadrise, sheer, transom) as a FreeCAD Body
- [x] 002 — export-module — full track — Deterministic .FCStd / STEP / STL / BREP writers with byte-identical reproducibility
- [x] 003 — deck-module — full track — Deck plate, cabin trunk, windshield, hardtop, railings mounted on the hull
- [x] 004 — interior-module — full track — Cabins, galley, heads, salon driven by canonical Alternativ1–5 YAML fixtures
- [x] 005 — cli-module — full track — `storebro build/list-layouts/info` CLI composing hull + deck + interior + export
- [x] 006 — partdesign-hull-upgrade — full track — Refactor hull from `Part::Loft` + `Part::Mirroring` into a clean `PartDesign Body` with editable `PartDesign::AdditiveLoft` + `PartDesign::Mirrored` feature stack (closes v0.1.0-alpha caveat from spec 001)
- [x] 007 — hull-fidelity-refresh — full track — Re-derive hull silhouette to match the storebropassion.de RC34 1972 reference (`docs/references/Alternativ3.JPG` profile): blunt stem, near-vertical stem rake (~6°), reduced sheer differential, smooth `Ruled=False` loft now that the stem is non-degenerate, plus optional chine. Visual fidelity over alpha-grade wedge.
- [x] 008 — deck-superstructure-refresh — full track — Reshape cabin trunk, windshield, hardtop, hardtop pillars, and railings to match the Alternativ1–5 reference profiles (currently boxy placeholders). Fix the hardtop-pillars-drop-below-hull positioning bug. Largest single visual win — superstructure is ~60% of what the eye reads as "Storebro".
- [x] 009 — hull-surface-smoothness — full track — Bump station count from 5 to ≥8 so `PartDesign::AdditiveLoft` Ruled=False (B-spline) converges without overshoot. Unblocks all three deferred markers from spec 007: `Hull.b_spline_loft`, `Hull.bilge_arc` (quarter-circle bilge transition), and `Hull.stem_with_zero_forefoot` (drop the 80mm virtual forefoot now that mixed topology is no longer required). Faceted between-stations look disappears.
- [x] 010 — deck-hardware-basic — full track — Add rubrail (wooden strip along the sheer — the single most identifying detail on a vintage Storebro), bow pulpit, lifelines, anchor locker, and basic cleats. Small geometric additions, large perceptual return.
- [x] 011 — window-cutouts — full track — Boolean cuts for portholes in the hull, transparent windshield + cabin trunk side windows. Currently the windshield is a grey plate; should read as glass with a frame.
- [x] 012 — interior-detail-alternativ1-2 — full track — Replace the boxy Alternativ1 + Alternativ2 placeholders with proper bulkheads, berths with cushions, galley with sink/stove/counter cutouts, heads with toilet/sink, salon with seating + table. Reference: `docs/references/Alternativ1.JPG`, `Alternativ2.JPG`.
- [x] 013 — interior-detail-alternativ3-5 — full track — Same treatment for Alternativ3, 4, 5. Splitting from spec 012 keeps PR scope sane; the canonical layouts share structural patterns so the second batch leverages spec 012's helpers.
- [x] 014 — propulsion — full track — Engine bay, engine block, shaft, propeller, rudder. Currently slated as v1.1+ per PROJECT-BRIEF; promote when the hull + deck + interior are solid enough that the boat needs to actually go somewhere.
- [ ] 015 — render-attributes — spec-only track — Set `App::PropertyColor` and `App::PropertyMaterial` on each Body so the GUI renders gelcoat-white hull, teak-brown trim, chromed railings, glass windows. No new entities, no state transitions — pure cosmetic property assignment.
- [ ] 017 — interior-scale-fix — spec-only track — Fix the interior module's meter-magnitude `Part.makeBox` convention (a 2.4 m cabin currently builds as a 2.4 mm box, ~1000× smaller than the mm-scale hull) so compartments + spec 012/013 furniture sit at true scale inside the hull. Discovered during spec 012 (see register history 2026-05-29). Pure scale correction — no new entities/state.
- [ ] 016 — ds-variant-superstructure — light track — Add a second canonical superstructure variant matching the DS model (built-in styrhytt / enclosed wheelhouse) seen in `docs/references/storo34_side_lines.png`. Reuses the hull from spec 007 verbatim — DS and standard RC34 share the underwater shape; only the deckhouse differs. Layout-switching infrastructure already exists from interior Alternativ1–5.
